## О проекте

Этот проект разрабатывался для Калужского городского хаккатона, прошедшего 20.10.2020. С этим проектом моя команда заняла 2е место. Разработка заняла шесть часов. Проект представляет собой Интерактивный онлайн квест. В разработке не использовалось БД, для меньших затрат времени.
